# 図鑑機能デザイン

## 構想
- 図鑑画面では、敵の一覧を表示する。
  - enemies.tsの敵データをもとに情報を表示する。
- 敵には，未発見，発見済み，撃破済み，外見解禁済みの4段階の状態がある。
- 画面左半分には以下の要素を表示する。
  - 戻るボタン
  - 敵表示パネル
    - レア度
    - アイコン
    - 名前
    - 説明文
    - HP
    - EXP, GOLD
    - 使用スキル
  - 状態パネル/ボタン
    - 未発見：グレーアウト「NOT FOUND」
    - 発見済み：グレーアウト「DEFEAT TO UNLOCK MOAI」
    - 撃破済み：「???G TO UNLOCK MOAI」※外見解禁要素は未定なので仮
    - 外見解禁済み：「USE THE MOAI」
- 画面右半分には、以下の要素を表示する。
  - ソート/フィルタリングオプション
    - レア度順、名前順などのソートオプション
    - 発見済み/撃破済み/外見解禁済みのフィルタリングオプション
  - 敵一覧
    - スクロール可能な形で並べて表示する。
      - グリッドかリストかはボタンから切り替えられるようにする。
    - 敵は、レア度の高い順に並べる．
    - 未発見の敵は、グレーアウト「???」として表示する。
- 敵の状態は、プレイヤーがその敵を発見したり撃破したりすることで変化する。
  - 開発段階では，すべての敵が発見済み/撃破済み/外見解禁済みの状態で表示されるようにしたい．
  - localStorageなどを使用して、敵の状態を保存する想定。※後でよい
