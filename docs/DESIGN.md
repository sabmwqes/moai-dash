# Moai Dash デザインガイドライン

- ダッシュボードのように。一画面で完結。
  - コンポーネントを配置していく感じ．
- 真っ黒を基調。ダークではなく#000000．
- ハイコントラスト。枠や文字は基本白．差し色として高彩度の色を差し込む。
- border-radiusは使わない！ハッカーのような効率重視でかっこいいデザインで。
- 横長の画面前提。ある程度のアスペクト比(3:4くらい)からはレスポンシブ対応しない。横のスクロール発生させてでも横長UIを維持する。
  - app-viewportでアスペクト比を維持している．絶対にはみ出してはいけない．
- 余白などはpxではなくremで指定する．
- 基本的に英語.
