# Moai Dash デザインガイドライン

- ダッシュボードのように。一画面で完結。
  - パネルコンポーネントを配置していく感じ．
- ハッカーのような効率重視でかっこいいデザインで。
  - 真っ黒を基調。ダークではなく#000000．
  - ハイコントラスト。枠や文字は基本白．差し色として高彩度の色を差し込む。
  - border-radiusは使わない！
- 横長の画面前提。ある程度のアスペクト比(3:4くらい)からはレスポンシブ対応しない。
  - app-viewportでアスペクト比を制限している．
- 余白などはpxではなくremで指定する．border-widthはpxで指定する．
- 基本的に英語.
